# To-do Manager

## Project description

This project is a To-do Manager application built with Vue.js and Vuetify. It allows users to create, read, update, and delete to-do items. Its interface is intuitive and user-friendly.

### Project technologies

- Vue.js
- Vuetify
- TypeScript
- Vite
- ESLint
- Prettier

## Project setup

If you want to start the project, you will need to install the different dependencies contained in the package.json file.
In order to do that, you will need to run the following command:
```sh
npm install
```

Once the dependencies are installed, you can start the project by running the following command:
```sh
npm run dev
```

It will start on http://localhost:5173.
